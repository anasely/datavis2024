Visualisation des séries avec le nombre d'épisodes, et le genre de la série avec l'utilisation de Rawgraphs :

Requête pour récupérer des informations sur des séries télévisées :

      SELECT ?serie ?serieLabel ?image ?creator ?actor ?composer ?country ?release_date ?nombreEpisodes ?dateDebut ?genreLabel ?scenaristeLabel
      WHERE {
        ?serie wdt:P31 wd:Q5398426;
          wdt:P18 ?image;
          wdt:P170 ?creator;
          wdt:P161 ?actor;
          wdt:P86 ?composer;
          wdt:P495 ?country;
          wdt:P577 ?release_date;
          wdt:P1113 ?nombreEpisodes;
          wdt:P136 ?genre;
          wdt:P58 ?scenariste.
          
        ?genre rdfs:label ?genreLabel.
        ?scenariste rdfs:label ?scenaristeLabel.
        
        FILTER(LANG(?genreLabel) = "fr" && LANG(?scenaristeLabel) = "fr")
        SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
      }
      ORDER BY DESC(?dateDebut)
      LIMIT 10
